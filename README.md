### [Go Programming Tutorial](./go)
> "Simplicity is the ultimate sophistication."

Explore 12 interactive modules covering the core of Go.
- **Topics**: Variables, Constants, Arrays, Slices, Maps, Structs, Defer, and more.
- **Features**: Interactive console menu, clean output, and practical examples.
- **Get Started**: `cd go && go run .`

### [Rust Programming Tutorial](./rust)
> "Empowering everyone to build reliable and efficient software."

Dive into 15 in-depth modules focusing on Rust's unique guarantees.
- **Topics**: Ownership, Borrowing, Lifetimes, Enums, Pattern Matching, Collections, and more.
- **Features**: Detailed explanations of memory safety, interactive examples, and clear visualizations.
- **Get Started**: `cd rust && cargo run`
